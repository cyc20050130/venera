# Changelog

## 2.0.0 - 2026-07-16

- 新增重写版升级保护流程，完成外部备份校验后再重置旧数据。

- 重构数据仓库与启动流程，提升备份恢复、压缩漫画重连和异常中断恢复的可靠性。

## 1.6.34 - 2026-07-16

## 1.6.30 - 2026-07-16

- Release tag: `v1.6.30`
- APK 架构：`arm64-v8a`

- 新增本地漫画无损 ZIP 归档：支持单本/批量压缩、阅读前自动恢复、保留 ZIP、再次压缩仅清理解压文件，并保持阅读历史、章节和页码进度。
- 增加归档原子提交、SHA-256 校验、安全路径检查、中断恢复和下载写入租约；修复重压缩遗漏旧章节、首次提交中断及归档状态删章后文件复活等数据一致性问题。
- 新增 Backup V2 逻辑快照和事务导入，可将设置页导出的 `.venera` 导入重写版数据库，同时继续兼容旧备份；保留历史、收藏、本地索引、设置、漫画源及缺失归档重连信息。
- 优化图片下载内存复制、阅读器解码尺寸、缓存上限恢复和事件驱动预取调度，修复同键请求竞态并限制后台预取并发。
- 接入 Riverpod、`sqlite_async`、`go_router` 和 Material 3 自适应设计系统骨架，升级 SQLite 运行时并移除已停止维护的旧兼容包。
- 增加归档、备份导入、路由、数据库、图片解码和预取调度回归测试；全量 525 项测试通过。
