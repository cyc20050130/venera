# Changelog

## 1.6.24 - 2026-05-24

- Release tag: `v1.6.24`
- APK 架构：`arm64-v8a`

- 修复阅读器刚进入漫画时中央点击经常拉不出底部工具栏的问题：reader 在加载阶段也会立即挂上手势层，不再等图片链完全就绪后才能响应点击。
- 修复 continuous 模式里“滚动结束后仍把点击当成滚动中”的回归，中央点击不再需要等一会或额外唤醒才能打开工具栏。
- 保留前一轮图片加载优先级、预取和详情导入后的可测环境，不回退已验证有效的在线看图优化。

## 1.6.23 - 2026-05-24

- Release tag: `v1.6.23`
- APK 架构：`arm64-v8a`

- 将阅读器工具栏与点击手势链回滚到 `1.6.19` 的稳定行为，撤掉后续因误判 SDK 三键问题而引入的工具栏点击分流逻辑。
- 恢复连续模式滚动后的旧式点击抑制时序，修复中央点击经常拉不出底部工具栏的回归。
- 保留图片加载优先级与预取优化，不回退在线看图缓存和当前章优先的性能修复。

## 1.6.22 - 2026-05-24

- Release tag: `v1.6.22`
- APK 架构：`arm64-v8a`

- 修复阅读器里“翻页区点击在防误翻冷却期被直接吃掉”的问题：这类点击现在会回退为打开工具栏，不再出现点了没反应。
- 保留真正的边缘误翻保护，但不再把被 suppression 挡下来的点击变成静默 no-op。
- 补充对应手势回归测试，覆盖 suppression 下 edge tap 回退为工具栏打开的行为。

## 1.6.21 - 2026-05-24

- Release tag: `v1.6.21`
- APK 架构：`arm64-v8a`

- 修复连续模式阅读器里点击漫画中央仍拉不出底部工具栏的问题：滚动刚结束的冷却期不再被误判成“仍在滚动”，中央点击现在会正常传递到工具栏开关。
- 保留边缘区域点按翻页的防误触抑制，不回退上一轮已经生效的 reader 图片优先级与缓存命中修复。
- 补齐连续模式点击链的回归验证，避免工具栏唤起再次被滚动状态误伤。

## 1.6.20 - 2026-05-24

- Release tag: `v1.6.20`
- APK 架构：`arm64-v8a`

- 修复在线看漫画时运行时缓存命中后仍继续偷偷发网络请求的问题，减少连续几张图变慢、尤其最后一张更慢的回归体感。
- 将阅读器图片任务拆成当前可见、同章预热、下一章预热三档优先级，确保当前章读图优先，下一章预热在有同章待热图时自动让路。
- 修复阅读器中央点击经常拉不出底部工具栏的问题，解耦“防误触翻页”和“禁止打开工具栏”的 suppression，滚动/翻页后中央点击可稳定拉起工具栏。
- 补充阅读器图片缓存策略、预热调度与工具栏点击逻辑的回归测试。

## 1.6.19 - 2026-05-24

- Release tag: `v1.6.19`
- APK 架构：`arm64-v8a`

- 保留阅读器章节切换重置和 `Overlay` 重建修复，不回退上一轮已经稳定下来的切章与重叠问题修复。

## 1.6.18 - 2026-05-23

- Release tag: `v1.6.18`
- APK 架构：`arm64-v8a`

## 1.6.17 - 2026-05-22

- Release tag: `v1.6.17`
- APK 架构：`arm64-v8a`

- 修复详情页进入阅读器时偶发的详情页与阅读器重叠闪现，阅读器入口改为同栈导航并禁用该链路的 route snapshotting。

## 1.6.16 - 2026-05-22

- Release tag: `v1.6.16`
- APK 架构：`arm64-v8a`
